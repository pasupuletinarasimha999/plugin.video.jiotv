Install

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://pasupuletinarasimha999.github.io/plugin.video.jiotv/ (Give it the name "TobAlan Repo").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "TobAlan Repository", and install repository.tobal-2.0.1.zip.
  • Restart Kodi
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "Tobalan Repository"
  • Go into the "Video add-ons" section in the repo, and you'll find all video add-ons
